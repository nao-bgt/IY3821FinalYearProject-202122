Implementation of a Binary to RGB Image Converter as a Class Object.

This is based on Necmettin Çarkacı's implementation of a Binary to RGB/Grayscale Image Converter (https://github.com/ncarkaci/binary-to-image/blob/master/binary2image.py) and L. Nataraj's work (http://dl.acm.org/citation.cfm?id=2016908).
